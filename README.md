# Dandle

A 3D word-building physics game. Build crossword-like structures with letter cubes on a checkerboard floor. Verbs apply directional force - use them to push your word structure into the end zone.

## How to Play

1. A random word appears as 3D letter cubes
2. Click a letter cube to select it
3. Type a word containing that letter and press Enter
4. The new word branches off perpendicular, Scrabble-style
5. **Verbs apply force** in the direction they're placed (indicated by red arrows)
6. Guide your word structure to the red end zone to complete the level

## Controls

- **Left click drag** - Orbit camera
- **Scroll** - Zoom
- **Right click drag** - Pan
- **Left click** (no drag) - Select a letter cube
- **Type + Enter** - Place a word

## Tech

- Three.js (3D rendering)
- Procedural audio (Web Audio API)
- No build step - static files, hosted on GitHub Pages
